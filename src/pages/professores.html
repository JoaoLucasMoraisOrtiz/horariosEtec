Cadastro de Professor(a)

Todos os professores devem ser cadastradas, para isso basta clicar no botão "Cadastrar Professor" e preencher todos os dados.

Exemplo:
Nome do Professor: Jorge Ribeiro dos Santos
Apelido da Professore: Jorge
Matéria que Leciona: Matématica
...

Registrar Horários

Para determinar quais aulas os professores já cadastrados desejam pegar basta clicar no botão "Horários" e você sera redirecionado para uma página em que você deve escolher a sala que você deseja determinar os horários e quais professores vão lecionar para aquela turma, após isso os professores com maior pontuação serão chamados para registrar os horários que querem lecionar para aquela turma, levando em consideração os hórarios já ocupados.
